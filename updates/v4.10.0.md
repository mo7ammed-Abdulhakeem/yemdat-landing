# v4.10.0 — Filament admin UX (nav groups, language toggle, table filters) (2026-06-02)

Usability pass over the Filament admin panel.

## Navigation grouping + icons
Resources/pages are now grouped in the sidebar (with distinct icons) instead of one flat list:
- **Community** — Members, Trainer Requests, Contacts, Certificates
- **Content** — Events, Posts, Broadcasts, Email Templates
- **Settings** — Membership Tiers, Users, Certificate Template, Settings
Group order is fixed via `navigationGroups()`; each item has a `navigationSort`.

## Language / direction toggle
A language button in the top bar (`PanelsRenderHook::TOPBAR_END` → `filament.lang-switcher`)
switches EN ⇄ AR via the existing `/lang/{locale}` route. `SetLocale` already feeds the panel, so
the whole admin flips **LTR ⇄ RTL** (Filament's own strings localize too). Removed the noisy
per-request `Log::info` calls from `SetLocale`.

## Collapsible sidebar on desktop
`->sidebarCollapsibleOnDesktop()` — the burger now collapses/expands the sidebar on large screens,
not just mobile.

## Tables: newest-first + filters/sorts
Consistent ordering and real filters across the list pages:
- **Members** (shipped in v4.9.3): newest-first; filters for membership, gender, verified, subscribed.
- **Contacts**: newest-first; sender filter (registered members vs guests); cleaner columns.
- **Trainer Requests**: newest-first; program-type + "agreed to free provision" filters.
- **Posts**: newest-first; type / published / featured filters; image + bilingual title columns.
- **Events**: start-date desc; active filter; trimmed columns (title + AR description, lecturer).
- **Certificates**: status filter (valid/revoked) — already issued-date desc.
- **Broadcasts**: status + audience filters — already created desc.

## Files changed (deploy to production)
- `app/Providers/Filament/AdminPanelProvider.php`, `resources/views/filament/lang-switcher.blade.php` (NEW)
- `app/Http/Middleware/SetLocale.php`
- All `app/Filament/Resources/*/{Resource}.php` (navigationGroup/icon/sort) + `app/Filament/Pages/{ManageSettings,ManageCertificateTemplate}.php`
- Table classes: Contacts, TrainerRequests, Posts, Events, Certificates, EmailBroadcasts

No migrations, no new dependencies, no asset rebuild (Filament assets unchanged). 109 tests green.
